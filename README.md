# improved-dollop
**Installing and Using OpenVAS for Vulnerability Scanning on Windows and Linux Systems**
This project demonstrates the identification, exploitation, and mitigation of OS Command Injection vulnerabilities in Linux-based web applications.
Implanting ethical hacking techniques, the objective is to scan for vulnerabilities, perform command injection attacks, and analyze the risks associated with insecure input handling.
